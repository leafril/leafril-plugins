## PRD

### 배경

bilgex는 WoW 경매장 시세를 추적하는 서비스다. 현재 사용자가 특정 아이템의 시세를 확인하려면 매번 검색해야 한다. 관심 아이템이 여러 개일 경우 반복 검색이 번거롭고, 시세 변동을 한눈에 비교하기 어렵다. 즐겨찾기 기능을 통해 관심 아이템을 등록해두고 시세 변동을 모아볼 수 있게 한다.

### 요구 사항

- 사용자가 경매 아이템을 즐겨찾기에 추가/제거할 수 있다.
- 즐겨찾기한 아이템 목록을 조회할 수 있다.
- 즐겨찾기 목록에서 각 아이템의 현재 시세 요약(최저가, 평균가, 수량)을 함께 확인할 수 있다.
- 즐겨찾기 목록에서 각 아이템의 시세 변동(전일 대비 가격 변화)을 확인할 수 있다.
- 즐겨찾기 목록에서 특정 아이템을 선택하면 기존 상세 시세 조회로 진입할 수 있다.

### 정책

- 즐겨찾기는 아이템 + 보너스 조합 단위로 등록한다.
- 사용자당 즐겨찾기 등록 수는 최대 50개로 제한한다.
- 동일한 아이템 + 보너스 조합을 중복 등록할 수 없다.
- 시세 변동은 직전 동기화 시점 대비 최저가 기준으로 계산한다.

### 기대 효과

- 관심 아이템의 시세를 반복 검색 없이 한 화면에서 모니터링할 수 있다.
- 시세 변동을 한눈에 파악하여 매수/매도 타이밍 판단에 활용할 수 있다.

| 시세 변동 계산 | 0.5d | | | | 전일 대비 가격 변화 |

### 의존 관계
- 즐겨찾기 목록 조회는 등록/제거 API 이후 진행한다.
- 시세 변동 계산은 기존 시계열 시세 조회 API의 데이터를 활용한다.

### 일정
- 낙관: 2d
- 현실: 3d
- 보수: 4d

### 리스크
- 사용자 식별 체계가 아직 없다면 인증/세션 설계가 선행되어야 한다. 즐겨찾기는 사용자별 데이터이므로 사용자 구분이 필수적이다.
- 즐겨찾기 목록 조회 시 아이템별 시세를 함께 반환해야 하므로, 아이템 수가 많을 경우 응답 시간이 길어질 수 있다.

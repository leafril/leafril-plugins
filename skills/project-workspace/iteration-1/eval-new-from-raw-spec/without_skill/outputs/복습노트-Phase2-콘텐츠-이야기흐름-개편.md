## PRD

### 배경

복습노트 Phase1은 보호자에게 일방적으로 제공되는 리포트 형태로, 감정 선택이나 복습 콘텐츠 열람이 제대로 이루어지지 않고 있다. 보호자가 가정에서 아이와 함께 복습하는 경험을 통해 교육적 가치를 체감하게 하고, 아이가 감정 표현과 보상(복습 도장)을 동기로 다음 활동 기록 촬영에도 능동적으로 참여하게 만들기 위한 Phase2 개편이다.

핵심 목적은 "안 하면 손해"라고 느낄 정도의 즉각적인 변화와 보상을 제공하여 복습률(도장 획득률)을 높이는 것이다.

별 시스템, 알림톡 이원화, 사진 복수화는 [[오늘찰칵-별시스템-알림톡-재설계]]에서 다룬다. 이 문서는 복습 콘텐츠 도메인 신규 구축과 이야기 흐름 개편을 다룬다.

### 요구 사항

**추가**

- **복습 콘텐츠** — 관리자가 반+날짜별로 생성하는 학습 콘텐츠. 활동 제목, 핵심 내용, 활동 요약(1-3문장), 가정 질문(1-2문장)으로 구성. 같은 반의 모든 원아 활동 기록에 공통 적용.
- **복습 콘텐츠 CRUD** — 관리자용 복습 콘텐츠 생성/수정/삭제/조회 API.
- **이야기 완료 후 콘텐츠 제공** — 완료 상태에서 질문&응답, 복습 콘텐츠, 활동 사진 다운로드를 함께 반환.
- **TTS 지원** — 복습 콘텐츠의 듣고 말하기 기능을 위한 음성 데이터 제공.

**변경**

- **이야기 흐름 개편** — 기존 스크롤 방식에서 페이징 방식(인트로 → 감정 선택 → 자유 기록 → 아웃트로)으로 전환. 각 단계 상태를 관리하여 중간 이탈 후 재진입 시 이어가기 지원.
- **상세 조회 확장** — 완료 상태일 때 복습 콘텐츠(활동 요약, 가정 질문)를 함께 반환.
- **활동 사진 다운로드** — 이야기 완료 후 원본 활동 사진 다운로드 URL 제공.

### 정책

**복습 콘텐츠 생성** — 관리자가 반과 날짜를 지정하여 생성. 해당 반+날짜의 모든 원아 활동 기록에 자동으로 연결된다.

**복습 콘텐츠 삭제** — 콘텐츠 삭제 시 연결된 활동 기록의 콘텐츠 참조만 해제. 활동 기록 자체는 유지.

**이야기 흐름 단계** — 인트로 → 감정 선택 → 자유 기록 → 아웃트로. 감정 선택은 필수, 자유 기록은 선택. 아웃트로 도달 시 이야기 완료 처리.

**활동 사진 블러 해제** — 이야기 진행 단계에 따라 활동 사진이 점진적으로 선명해진다. 블러 단계 정보를 API에서 제공.

**TTS 콘텐츠** — 복습 콘텐츠의 활동 요약과 가정 질문에 대해 음성 데이터를 제공. 음성 생성 방식(사전 생성 vs 실시간)은 구현 시 결정.

### 기대 효과

- 복습 콘텐츠로 가정에서의 학습 연계 → 보호자의 교육적 가치 체감
- 이야기 흐름 개편(페이징 + 블러 해제)으로 참여 몰입도 향상 → 이야기 완료율 상승
- TTS로 아이가 직접 듣고 말하는 경험 제공 → 참여 경험 개선
- 성공 지표: 복습 콘텐츠 열람률, 이야기 완료율, TTS 사용률

### 리스크

- **TTS 연동 (1.5d)**: 음성 생성 방식(사전 생성 vs 실시간 TTS API)에 따라 공수 차이가 큼. 외부 서비스 의존 시 비용·응답 시간 검증 필요.

- **블러 단계 정책**: 프론트엔드 블러 처리와 백엔드 단계 정보의 동기화. 정책 변경 시 양쪽 수정 필요.

# 복습노트 Phase2 기획

### 배경
- 현재 복습노트 phase1은 보호자에게 일방적으로 제공되는 리포트 UI에 가까워 감정 선택이나 복습 콘텐츠 보기 등이 제대로 진행되고 있지 않음.
- 보호자가 가정에서 실제로 복습을 하는 아이의 모습을 보고 교육적 가치를 느끼게 하기 위함.
- 아이가 감정을 선택하고 보상으로 받는 복습도장이 동기부여가 되어 다음 복습노트 촬영도 능동적으로 할 수 있도록 유도하기 위함.

### 목적
- 아이와 보호자가 안 하면 손해라고 느낄 정도로 즉각적인 변화/보상 등을 제공하여 확실히 행동하게 한다.
- 복습률(도장획득률)을 올린다.

### 참고 자료
- 웅진스마트올 앱 분석 (출석 스티커판, 학습 완료 보상, 기프티콘 교환 등)

### 가설
- 하원 이후 시간대에 열람 가능하게 하고 알림을 보내면 당일 복습률이 증가할 것이다.
- 스티커판 UI로 복습 완료율을 시각적으로 보여주면 복습률이 증가할 것이다.
- TTS로 듣고 말하기 기능을 제공하면 참여 경험이 개선될 것이다.
- 활동 사진을 블러 처리하고 단계별로 선명해지게 하면 복습 집중률이 올라갈 것이다.

### 정책
- 오늘찰칵(활동 사진 촬영 시 자동 생성, 원아별/날짜별)
- 복습 콘텐츠(관리자가 생성, 반+날짜별, 여러 원아의 오늘찰칵에 공통 적용)
- 오늘찰칵 생성: 원아당 하루 1장, 활동 사진 저장 완료 시 생성, 오후 17시부터 노출
- 오늘찰칵 삭제: 활동 사진 삭제 시 함께 삭제
- 복습 콘텐츠 구성: 날짜, 대상 반, 활동 제목, 핵심 내용, 활동 요약(1-3문장), 가정 질문(1-2문장)
- 별 시스템: 이야기 전=회색별, 당일 완료=황금별, 당일 이후 완료=일반별

### 요구사항
- 발행 시간 설정: 당일 촬영 사진은 17시부터 노출
- 알림톡 발송 조건 변경:
  - 발송 시간: 20시 27분
  - 조건A(온보딩): 이야기 완료 한 번도 안 한 kid 대상, 최근 30일 이내 알림톡 미수신
  - 조건B(리마인드): 최소 1회 이상 완료한 kid 대상, 최근 3건 중 1건 이상 완료, 오늘은 아직 미완료
- 홈 UIUX 개선: 카드 리스트뷰 → 별자리 UI (별 상태 3종)
- 상세 UIUX 변경: 스크롤 → 페이징 (인트로→감정선택→자유기록→아웃트로)
- 이야기 완료 시 별 애니메이션 + 효과음
- 이야기 완료 상태에서 질문&응답, 복습 콘텐츠, 사진 다운로드 제공
- 도움말 페이지 추가

### 화면 구조
- 플러스: 오늘찰칵 리스트 → 오늘찰칵 상세 (이야기 전/후)
- 플레이: 반 선택 → 모드 선택 → 원아리스트 → 카메라/저장된 사진
- 각 화면별 오디오 에셋, 애니메이션 스펙 상세 존재 (생략)

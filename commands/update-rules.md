# Update Rules

노트 루트의 CLAUDE.md 작성 규칙을 업데이트하는 서브커맨드입니다.
세션 중 발견한 규칙 개선사항을 반영할 때 사용한다.

## 실행 프로토콜

### 전제조건: 노트 루트 경로 확인

`~/.claude/note-plugins.json`에서 `noteRoot` 값을 확인한다. 파일이 없으면 새로 생성한다. `noteRoot`가 없거나 존재하지 않는 경로이면 AskUserQuestion으로 물어보고, 응답을 저장한다.

### Phase 1: 현재 규칙 파악

1. 노트 루트의 CLAUDE.md를 읽고 현재 규칙을 파악한다.

### Phase 2: 변경 사항 반영

1. 사용자가 전달한 내용에 따라 규칙을 추가하거나 수정한다.
2. 기존 규칙과 충돌하는 내용이 있으면 사용자에게 확인한다.

### Phase 3: Git 커밋

작성 완료 후 자동 커밋한다.

## 주의

- 구조화된 질문으로 사용자를 유도하지 않는다.
- 사용자가 말한 내용을 그대로 반영한다. 임의로 규칙을 추가하거나 해석을 넣지 않는다.
- 기존 규칙의 의도를 훼손하지 않는다.
